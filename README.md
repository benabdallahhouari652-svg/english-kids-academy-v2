## <p align="right">📋 حول المشروع</p>
<p align="right" dir="rtl">
<b>English Kids Academy</b> هي منصة تعليمية تفاعلية مخصصة لتعليم اللغة الإنجليزية للأطفال الناطقين بالعربية. 
تقدم المنصة محتوى تعليمي مبسط وممتع يساعد الأطفال على تعلم الإنجليزية بطريقة طبيعية من خلال الألعاب والصور والصوتيات.
</p>

## <p align="right">✨ المميزات</p>
<p align="right" dir="rtl">
- 🎮 دروس تفاعلية مع ألعاب تعليمية<br />
- 🔤 تعليم الحروف والكلمات الأساسية<br />
- 🖼️ استخدام الصور والرسوم التوضيحية<br />
- 🎵 دعم الصوتيات والنطق الصحيح<br />
- 📱 واجهة متجاوبة مناسبة للأطفال<br />
- 🌐 واجهة باللغة العربية مع محتوى إنجليزي<br />
- 📊 تتبع تقدم الطفل في التعلم
</p>

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **Next.js 16** | إطار العمل الرئيسي |
| **TypeScript** | لغة البرمجة |
| **Tailwind CSS** | تصميم الواجهات |
| **Prisma 6** | إدارة قاعدة البيانات |
| **SQLite** | قاعدة البيانات |

## 🚀 كيفية التشغيل

```bash
# تثبيت الاعتماديات
npm install

# تشغيل وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# تشغيل النسخة الإنتاجية
npm start
```

افتتح [http://localhost:3000](http://localhost:3000) في المتصفح.
